# 반응형 이미지 인터랙션 프로젝트

이 프로젝트는 웹페이지 스크린샷 이미지를 활용하여 실제로 동작하는 것처럼 보이는 웹페이지를 구현한 것입니다. 화면 크기가 변경되어도 버튼의 위치가 이미지와 항상 맞춰지도록 설계되었습니다.

## 주요 기능

- 스크린샷 이미지 기반 인터랙티브 웹 시뮬레이션
- 반응형 디자인: 화면 크기가 변해도 버튼 위치가 이미지에 정확히 매핑됨
- 페이지 간 이동 기능으로 여러 화면 구현
- 개발자 모드를 통한 쉬운 버튼 영역 설정

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 화면에 표시된 버튼 영역을 클릭하여 다음 화면으로 이동합니다.
3. '이전'/'다음' 버튼으로 화면을 순차적으로 탐색할 수 있습니다.

## 개발자 모드

새로운 버튼 영역을 추가하고 싶을 경우:

1. 브라우저 콘솔에서 `enableDevMode()` 함수를 실행합니다.
2. 이미지의 원하는 위치를 클릭하면 해당 위치의 상대적 좌표(x, y)가 콘솔에 출력됩니다.
3. 이 좌표를 `script.js` 파일의 `screenMappings` 객체에 추가합니다.

## 추가 커스터마이징

- `styles.css`에서 디자인을 수정할 수 있습니다.
- `script.js`에서 버튼 영역과 화면 간 이동 로직을 변경할 수 있습니다.
